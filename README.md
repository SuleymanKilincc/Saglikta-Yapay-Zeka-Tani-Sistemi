# 🏥 Sağlıkta Yapay Zeka Destekli Tanı Sistemi

Bu proje, tıbbi görüntüleri (röntgen) analiz ederek hastalıklara (özellikle Pnömoni/Zatürre) erken teşhis koymada doktorlara yardımcı olmak amacıyla geliştirilen bir yapay zeka karar destek sistemidir.

## 💻 Kullanılacak Teknolojiler
* **Dil:** Python
* **Yapay Zeka:** TensorFlow / Keras (CNN Mimarisi)
* **Veritabanı:** PostgreSQL
* **Veri İşleme:** DICOM Kütüphaneleri
